# EXON

**EXON** is a cutting-edge face swapping and enhancement application designed for precision and performance. It features advanced masking (BiSeNet), granular control, and high-performance FP16 inference.

## Features

- **Face Swapping**: High-quality face replacement using InsightFace.
- **Advanced Masking**: BiSeNet integration with granular control (exclude eyes, mouth, etc.).
- **Performance**: Optimized for NVIDIA GPUs with FP16 support and tuned CUDA providers.
- **Enhancement**: Integrated face enhancers for crisp results with Granular "Enhancer Strength" control.
- **Visual Selector**: Intuitive gallery-based face selection.
- **Smart Presets**: One-click "Cinema", "Ultra", "Balanced" and "Fast" modes for optimized workflows.
